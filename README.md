free high-res templates for [einguteswerkzeug](https://github.com/s3h10r/einguteswerkzeug)
============================================

This repository provides free to use (free as in "freedom" - not as in "free beer")
**high-res templates** for `einguteswerkzeug`.

Contributions are welcome and they are greatly appreciated!
Please feel free to send a pull-request and use the [issue tracker](https://github.com/s3h10r/egw-templates/issues).
